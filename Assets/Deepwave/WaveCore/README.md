# Deepwave Core

**Deepwave Core** is a utility library for Unity that provides shared source code, mathematical utilities, design patterns, and custom attributes to enhance the Inspector experience and streamline development workflows.

## 📋 General Information
- **Version:** 1.0.6
- **Unity Compatibility:** 2021.3+
- **Author:** Thang Le
- **Namespace:** `Deepwave.Core`

---

## 🚀 Key Features

### 1. Attributes & Inspector Enhancement
This package includes attributes to professionally customize the Inspector interface:
- `[InspectorHeader("Title", icon = "icon_name")]`: Adds a stylized, horror-themed header (gradient background, glowing text) to any `MonoBehaviour`.
- `[DynamicRange(min, max)]`: Limits a float value with a custom slider.
- `[Vector2Range(min, max)]`: Allows inputting a range (Min/Max) on a single line.
- `[ProgressBar]`: Displays a visual progress bar for numerical variables.

### 2. Singleton Patterns
Provides safe and efficient Singleton implementations:
- `Singleton<T>`: Thread-safe, lazy-initialized Singleton for plain C# classes using `Lazy<T>`.
- `SingletonBehaviour<T>`: Singleton pattern specifically for Unity `MonoBehaviour` components.

### 3. Dynamic Types
Flexible data structures that allow switching between fixed values and randomized ranges:
- `DynamicFloat`: A struct containing a fixed value and a `Vector2Range`. Use `.Evaluate()` to retrieve either the fixed value or a random value within the range.
- `DynamicInt`: Similar to `DynamicFloat`, but designed for integers.

### 4. Math Utilities (`MathUtils`)
An extensive math library with useful functions:
- **Interpolation:** `FloatInterp`, `SmoothSlerp` (Uses Exponential Decay for smoother, framerate-independent movement compared to standard Lerp).
- **Rotation:** `FromToRotation`, `NormalizeSafe`, `ComputeLookAtInput`.
- **Angles:** `NormalizeEulerAngle`, `TriangleAngle`.
- **Logic:** `InvLerp`, `ExpDecayAlpha`.

### 5. Other Utilities
- `GuidUtility`: Utilities for generating and managing GUIDs.

## 🛠 Usage Examples

### Using Custom Inspector Header
```csharp
using Deepwave.Core;
using UnityEngine;

[InspectorHeader("Player Controller", icon = "d_UserControl")]
public class PlayerController : MonoBehaviour 
{
    [DynamicRange(0, 100)]
    public float speed;
    
    public DynamicFloat jumpForce;

    void Update() 
    {
        // Automatically returns a random value if 'randomize' is checked in the Inspector
        float currentForce = jumpForce.Evaluate();
    }
}
```

### Using Singleton
```csharp
public class GameSettings : Singleton<GameSettings> 
{
    private GameSettings() { } // Private constructor to ensure Singleton integrity
    public float Volume = 1.0f;
}

// Accessing the instance
float vol = GameSettings.Instance.Volume;
```

### Using MathUtils
```csharp
float currentHeight = 0f;
float targetHeight = 10f;

void Update() 
{
    // Framerate-independent interpolation
    currentHeight = MathUtils.FloatInterp(currentHeight, targetHeight, 5f, Time.deltaTime);
}
```
